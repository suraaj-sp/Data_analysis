#  Data Analysis & Scientific Computing Practice

This is my personal practice workspace for learning data analysis and numerical computing with Python.

## Folder overview

### Pandas Practice
- `pandas/` contains my practice workspace with notebooks and example scripts for data manipulation.
- `pandas/data/` stores the raw and exported car sales datasets.
- `pandas/analysis.py` is a practice script for loading and previewing the dataset.
- `pandas/pandas-01.ipynb` is an interactive notebook for exploration.
- `pandas/README.md` documents the pandas practice folder structure and usage.

### NumPy Practice
- `numpy/` contains my practice workspace with notebooks for numerical computing.
- `numpy/NumPy-1.ipynb`, `numpy/NumPy-2.ipynb`, `numpy/NumPy-3.ipynb` are practice notebooks.
- `numpy/analysis.py` is a practice script for array operations and computations.
- `numpy/README.md` documents the numpy practice folder structure and usage.

## Getting started

### Pandas:
```bash
cd pandas
pip install -r requirements.txt
python analysis.py
```

### NumPy:
```bash
cd numpy
pip install -r requirements.txt
python analysis.py
```
