# Personal NumPy Practice

This folder is a personal practice workspace for learning `numpy` and working with numerical arrays.

## Practice structure

```
├── NumPy-1.ipynb                 # First NumPy practice notebook
├── NumPy-2.ipynb                 # Second NumPy practice notebook
├── NumPy-3.ipynb                 # Third NumPy practice notebook
├── analysis.py                   # Example script for practice
├── requirements.txt              # Practice dependencies
└── README.md                     # This practice documentation file
```

## Usage

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the practice analysis:

```bash
python analysis.py
```

## Notes

- This is a local practice folder for experimenting with `numpy`.
- Open the notebooks for interactive numerical computing and array manipulation.
